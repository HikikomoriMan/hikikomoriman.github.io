Tentang Saya

Perkenalkan, nama saya Muhammad Rhifaldy Bhimantara Sopadi Putra, sering dipanggil Bhima. Saya lahir di Sumedang, 04 Mei 2004. Saya tumbuh dewasa diberbagai kota yaitu Jakarta, Cirebon, Palembang, dan Sumedang.

Saya memiliki beberapa hobi seperti renang, voli, dan bermain game. Sejak kecil saya sangat menyukai renang, dan bermain game. Saya mulai menyukai voli semenjak saya duduk dibangku SMP.

Saya mulai bisa renang sejak saya berumur 6 tahun, ketika teman sebaya saya masih banyak yang belum bisa berenang. Saya mulai bermain game dari sebelum saya masuk TK, saya sudah terbiasa bermain game di Playstation 2. Saya mahir bermain voli ketika saya duduk dibangku SMP. Pada saat itu desa saya memiliki organisasi voli, saya pun tertarik untuk mengikutinya.

Pada saat saya masih kecil saya memiliki cita-cita sebagai tentara. Namun, ketika saya mulai beranjak dewasa, saya berubah pikiran karena hobi saya bermain game. Sehingga saya memiliki ketertarikan memulai karir dalam pembuatan sebuah video game.

Ketika saya TK, saya tinggal di Jakarta. Pindah ke Cirebon ketika menginjak bangku SD. Selama 3 tahun di Cirebon, saya pindah lagi ke Palembang. Hanya setahun di Palembang saya kembali lagi ke tempat saya lahir yaitu Sumedang sampai saya lulus SMA. Setelah lulus SMA, saya memutuskan untuk kembali ke Jakarta untuk melanjutkan pendidikan saya di kampus ITTJ.
